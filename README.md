# Тестовое задание: Тестирование логики начисления кэшбэка (банковская тематика)

## Описание

Этот репозиторий содержит результаты тестового задания по проверке функциональности начисления, учёта и выплаты кэшбэка по банковским картам.

---

## Содержимое репозитория

- `report.md` — основной отчёт по тестированию:
  - чек-лист тестирования по настройкам кэшбэка (CASHBACK_TYPES, CASHBACK_EXCEPTION, CASHBACK_VALUE)
  - проверка условий начисления кэшбэка
  - проверка расчёта суммы кэшбэка (базовая и повышенная ставки, округление, лимиты)
  - сценарии учёта и выплаты кэшбэка
  - интеграционные (E2E) и негативные тесты
  - вопросы для уточнения у бизнес-аналитика

---

## Инструменты

- Аналитический разбор ТЗ (без доступа к тестовому стенду)
- Markdown для оформления чек-листов и отчёта

---

## Цель задания

Составить тест-кейсы и чек-лист для проверки корректности начисления, учёта и выплаты кэшбэка по банковским картам согласно техническому заданию, а также выявить неоднозначности и риски в требованиях.
